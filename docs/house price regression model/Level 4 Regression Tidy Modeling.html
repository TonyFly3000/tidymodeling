## workflow

```{r}
model_workflow =
  workflow() %>% 
  add_model(lasso_tune_spec) %>% 
  add_recipe(data_rec)
```

```{r}
cntl   <- control_grid(save_pred     = TRUE,
```

### train lasso model

```{r}
lasso_res = model_workflow %>%
```
